# Mini To-Do List

Simple interactive To-Do List built with vanilla JavaScript and DOM manipulation.

Features
- Add tasks
- Edit tasks inline (click text, press Enter to save)
- Mark complete / uncomplete
- Delete tasks
- Persist using `localStorage`

Run locally
1. Open `index.html` in your browser (double-click or use a local server).

Hosting (GitHub Pages)
1. Create a repo and push the project.
2. In GitHub repo settings enable Pages from the `main` branch (root).
3. Your site will be available at `https://<your-username>.github.io/<repo>`.

Screenshots
- Open the app in your browser and take screenshots of the main view and an edited task.

Files
- `index.html` — app UI
- `style.css` — styles
- `script.js` — app logic
